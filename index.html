如何使用

这是基于 Reasonix 制作的 Skill，建议 Reasonix-v1.3+ 以上版本。

如需在 Claude Code 等平台使用，发送指令附 esi-liuyao 文件即可适配。

~/app — ai chat
$ run skill esi-liuyao + 问题 + 事件背景

背景越详尽越好。回溯过去可附时间如 2022年2月22日22时22分。

重要声明

统计上有一定正确率，但我们不建议以此做任何决策。

我们不提供迷信服务。探究的是：符号排列组合后能否真实反映现实。

解读中难懂的词汇是包含巨量信息的符号——形式上的必须。我们去除神秘化，杜绝巴纳姆效应。不一定正确，也绝不宽泛。

Philosophy

三大核心支柱

01

Emperical · 实证

实践为证，每一条论断都经过评分验证。不做模糊正确的预测，只交付可度量的结果。

02

Systematic · 系统

覆盖出题、解读、评分、记录全流程。符号排列组合后，系统性逼近真实世界的答案。

03

Iterative · 迭代

不满足于单一版本的稳定。每个版本都是下一次的起点，在反馈循环中持续进化。
