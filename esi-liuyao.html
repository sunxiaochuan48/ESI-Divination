01 / 关于

关于这个项目

这是基于 Reasonix 进行制作的 Skill 项目，目的是以 AI 进行中国传统术数的实用性探究。如果您想使用它，在 Reasonix-v1.3+ 以上的版本运行最为稳妥。

Reasonix 官网入口 →

如果您需要在 Claude Code 等其他平台上使用，可向 AI 发送指令，并附带一份 esi-liuyao 文件，它会自动调整内容适应到当前平台。

如何使用

这是一个 六爻 Skill ，顾名思义，它可以做到从起卦到解读全过程。

使用时，只需要在任意 Ai 中输入以下内容即可

~/app — ai chat
$ run skill esi-liuyao + 问题 + 事件背景

关于背景，我们建议越详细越好，这会辅助它更快、更好的取象。

如果需要求问一件过去的事情，最好附带一串事情最初发生的时间，例如 2022年2月22日22时22分 ，这对判断有益。但如果你已经忘记了时间，也请不要勉强。

它会针对你的问题、卦象，给出合理的解读。虽然这在统计上拥有一定的正确率，但我们仍不建议用它来做任何决策与参考。这是对我们的保护，也是由衷建议，我们无法对它的判断与解读负责，这部分需要由你个人承担。

如何确保有效

我们拥有一整套训练系统，涵盖出题、解读、评分、记录等环节。在最初的版本中，它取得了69%的正确率，但这只能说明，它在部分场景上拥有一定的效果展现，这需要客观辩证的看待。

但更重要的，是我们的三大核心：实证、系统、迭代。

主张实证，保持诚实，以实践为证，从实践中获取经验与知识。我们不止制作Skill，在未来这个项目会长期开展，不断更新。敢于用数据丈量自身，不陷入模糊的正确中，是我们相较同类型Skill最大的优势。

我们不提供迷信

我们不提供迷信服务，也不会在此深耕探索。简单说，我们需要探究的是，符号经过排列组合后，最终能否真实、准确的反映现实。六爻这门术数，则是在权衡之下选择的载体。

在使用过程中，我们通常会看到一些难以理解的词汇。这本质是包含巨量信息的符号，一切的解读都是建立在一个又一个的符号之上。这并非神秘化，而是形式上的必须，抛去这一点，整套体系将无法运行。

我们会尽可能的去除神秘化元素，把东西摊开讲实。以及，杜绝巴纳姆效应的使用，让它成为指南针一样的存在，不一定正确，也绝不宽泛。即便指向错误，也无非是重新调整。
